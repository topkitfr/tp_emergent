<product_requirements>
TopKit aims to be a collaborative football jersey database, enabling users to estimate collection values and contribute to a community-validated dataset. Key entities include Team, Brand, Jersey (Release), Master Jersey, Player, and Competition. The platform requires a Discogs-like contribution system with visual proof, community voting, moderation, and version history. UI/UX should be consistent, with optimized image display, refined navigation, and enhanced search. The Master Jersey (design) and Jersey Release (physical version) distinction is central, with Le Vestiaire for managing Jersey Releases and integrated price estimation, alongside user collections (owned/wanted items).
