<product_requirements>
TopKit is envisioned as the most comprehensive collaborative database for football jerseys, functioning like a specialized wiki where users contribute and the community validates information.
The application's core purpose is to enable users to estimate the value of their jersey collections and contribute to a global, verified database.
Key entities include: Team/Club, Brand/Manufacturer, Jersey (Release), Master Jersey (grouping design versions), Player, and Competition/Event.
User contributions must adhere to strict rules, requiring mandatory visual proof (photos from all angles, tags, labels). A moderation system with community voting, moderators, and admins ensures data accuracy.
Each entity requires a unique identifier and standardized links between them. Technical details, version history, and handling of multi-variant jerseys are essential.
The platform aims to prevent duplicates through advanced search and detection algorithms, with a merging capability. A public API for data access and CSV/JSON export are required.
UI/UX mandates a new design theme, consistent branding (logo, background), optimized image views (square thumbnails, object-cover), a refined hamburger menu (no icons, red logout), and enhanced search. A Marketplace page should display Coming Soon. Social media links are limited to Instagram.
</product_requirements>
