<product_requirements>
TopKit is a soccer jersey marketplace enabling users to catalog, buy, and sell jerseys. Its core features include user profiles with owned and wanted collections, a searchable catalog, and a Discogs-like moderation model. The application has evolved to include an Explorez (Browse/Search) page, Jersey Detail pages with actions (Own, Want, Buy), Jersey Sale, and a global Marketplace. The UI features a consistent dark monochrome theme. A consolidated My Profile page integrates profile and collection views, displaying estimated collection value and tabbed sections (My Collection, My Wishlist, My Submissions). The application alerts users about duplicate jersey submissions.

Recent development, as detailed in the trajectory, focused on:
1.  **Friends Section in My Profile**: Displaying a user's friends with a count, and providing sub-tabs for pending friend requests and requests awaiting approval.
2.  **Anonymous Jersey Submission**: When a user adds a jersey, it should not be automatically attributed to their collection. Instead, it serves as a submission to populate the database for common use, subject to admin/moderator approval and correction. Users should be able to view their submissions and their status, and receive notifications upon approval or correction.
3.  **Enhanced Admin Moderation Interface**: Admins need the ability to retrieve and correct the original submission form filled by the user for a pending jersey. This includes a workflow allowing admins to Correct, Approve, or Delete submissions, with correction preceding approval.
