<product_requirements>
TopKit is a soccer jersey marketplace designed to mirror Discogs' functionality, allowing users to catalog, buy, and sell jerseys. Key functionalities include user profiles with owned/wanted collections, a searchable catalog, and a messaging system, all optimized for mobile with admin moderation. Initial development covered UI/UX improvements, comprehensive testing, fixing MongoDB serialization/UI state issues, and contact seller features. A Level 1 security overhaul (strong passwords, rate limiting, email verification, Google OAuth removal), Stripe payment integration (commission model), and a Private Beta Mode (with beta access requests and admin controls) were implemented. The core workflow clarifies: jersey submission for cataloging (no size/condition), admin approval, adding to personal collection (with size/condition), and then listing from collection. Recent work added a Moderator role, dynamic homepage statistics, integrated messaging/friends, and made notifications clickable.
</product_requirements>
